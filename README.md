# Jason Typing Game (Rust)

🦀 **Rust + iced で開発された、13日の金曜日（ジェイソン）テーマのデスクトップ・タイピングゲーム** 🪓

OSのネイティブフォントを自動認識する日本語対応の設計でありながら、出題されるワードはすべて恐怖のジェイソン（Friday the 13th）にちなんだカルト的な英語フレーズです。

---

## 🎮 機能特徴

- **クロスプラットフォーム対応 (iced)**: Windows, macOS, Linux で動作する軽量なGUIアプリケーション。
- **文字化け（豆腐文字）対策**: 起動OSを自動判別し、適切なシステム日本語フォント（遊ゴシック、ヒラギノ等）を自動適用。
- **精密なリザルト判定**: 正解数、入力ミス回数、ミリ秒単位での経過時間をリアルタイムに計測。
- **インテリジェントな入力制御**: 英単語内のスペース文字を考慮し、無駄な空エンターによるミス判定を排除。

📁 構造（システムアーキテクチャ）
本アプリケーションは、Elmアーキテクチャ（The Elm Architecture）にインスパイアされた iced のデータフローに基づき、安全かつ構造的に設計されています。

State (構造体): アプリケーションのすべての状態（タイマー、スコア、ミス、入力バッファ）を一元管理。

Message (列挙型): ユーザーのキー入力や確定イベントをタイプ安全にカプセル化。

Update (ロジック): メッセージ駆動型で、状態の変化に副作用をもたらさない厳密なデータ更新。

View (レンダリング): 更新された状態を元に、宣言的UIとして描画レイアウトを出力。
